### Desenvolvido com C

## Para rodar: 

Na pasta src, rode no terminal

```shell
make run
```

Este comando irá apagar o build atual, buildar o código usando o gcc e rodá-lo.

Você também pode fazer os passos individualmente, usando: 

```shell
make clean
make build
make run
```

Importante buildar antes do run, hein? 👀
